## Instrucciones de ejecución 
1. Descargar archivos del proyecto ubicados en el repositorio.
2. Abrir el proyecto en Apache NetBeans.
3. Ejecutar el archivo App.java para ver la aplicación en ejecución
4. Ingresar el nombre y la edad en consola cuando el sistema lo solicite (en caso de errores se mostrarán las salidas correspondientes).
5. Para visualizar el archivo en donde se enlistan los datos registrados ("usuarios.txt") puede acceder a la carpeta del proyecto ubicada en su ordenador.
